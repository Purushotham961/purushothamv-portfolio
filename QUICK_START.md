## 🎯 Deploy in 5 Minutes

### Quick 4-Step Process:

**1. Create GitHub Repo**
```
https://github.com/new
Repository name: Portfolio_AI
Make it Public
Click Create
```

**2. Clone & Upload Files**
```bash
cd c:\projects\New folder\Portfolio_Website
git init
git remote add origin https://github.com/Purushotham961/Portfolio_AI.git
git add .
git commit -m "Add portfolio website"
git branch -M main
git push -u origin main
```

**3. Enable GitHub Pages**
```
Go to: https://github.com/Purushotham961/Portfolio_AI
Settings → Pages → Branch: main / (root)
Save
```

**4. Your Site is Live!** ✅
```
🎉 https://Purushotham961.github.io/Portfolio_AI/
```
